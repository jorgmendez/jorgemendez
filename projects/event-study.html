Microsoft × Activision · Event Study

Tools: R, Excel · Windows: Estimation (2021-07-01 to 2022-01-10), Event (2022-01-11 to 2022-01-28)

Abstract

I evaluate market reaction to Microsoft’s $68.7B acquisition of Activision Blizzard announced on Jan 18, 2022. Using an estimation window to fit expected returns and an event window to measure impacts, I compute abnormal returns (AR) and cumulative abnormal returns (CAR). A two-sample t-test shows a significant mean shift; CAPM and a sector-adjusted multifactor model confirm significance. CAR increases ≈ +29% post-announcement.

Data

  • Sources: Yahoo Finance daily prices for ATVI and S&P 500.
  • Series: Close prices and computed daily log/percent returns.
  • Prep: Cleaned dates, aligned trading days, computed returns in R/Excel.
Returns aren’t perfectly normal, but with n ≈ 144 the t‑test is reasonably robust; CLT supports inference on mean differences.

Methodology

Steps: (1) Fit expected returns on the estimation window; (2) compute AR = RATVI − E[RATVI] on event days; (3) accumulate to CAR; (4) test significance.

  • t‑test: mean comparison (estimation vs event window returns).
  • CAPM: ri − rf = α + β (rm − rf) + ε
  • Multifactor: CAPM + sector index (ESPO) for gaming/esports exposure.

Results

Robustness

  • Winsorizing at 1%/99% leaves inference unchanged.
  • Percent vs log returns yield similar CAR shape.
  • Alternative windows (±3, ±5 days) preserve significance.
